# React新项目构建

## 项目结构

```
|- src/
    |- sass/
    |- images/
    |- js/
    |- js/utils/
    |- js/index.js
    |- routes/
|- dist/
|- index.html
|- gulpfile.js
|- node_modules/
|- packge.json
```

## 安装依赖

```shell
npm install
```

## 启动应用

```
gulp server
```

启动后在浏览器输入：`http://localhost:8080/`
